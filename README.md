# Projet_GBSC

Projet d’analyse comparative des départements français pour identifier les territoires offrant les meilleures conditions de vie, à partir de facteurs socio-économiques et culturels.

**Technologies utilisées :** Excel, R, RStudio, R Markdown, MySQL, phpMyAdmin, MAMP, Google Colab.  

**Résultats :** traitement et structuration des données, création de graphiques et visualisations, identification des principaux facteurs influençant le niveau de vie.  

**Projet réalisé en groupe de 4 personnes.**
